2. YUM (Yellodog Updater Modified) : its a wrapper around rpm, that can automatically resolve dependencies and retrive packages from repositories (repo).

read : r : view file content and list directories
write : w : edit the file content and create/delete files inside the directory
execute : x : run as program and enter the directory

Octal/Numberic Permissions Management method:

Read 	: r : 4
write	: w : 2
execute	: x : 1
None	: - : 0 

7 : rwx : 4+2+1 : Full Access
6 : rw- : 4+2+0 : Read and Write
5 : r-x : 4+0+1 : Read and Execute
4 : r-- : 4+0+0 : Read Only
0 : --- : 0+0+0 : No Access
