How you troubleshoot if your instance is unhealthy at targetgoup level. 
1. Instance is in running state
2. Verify instance security group, Required port is opned or not.!
3. Verify health check path you configured. make sure the path exists in your ec2 instances. 
4. Application running on configured port or not.!! If app is in stopped state, then no response to the traffic. 
5. Make sure your load balancer is running with all subnets, that your ec2 instances are running.
